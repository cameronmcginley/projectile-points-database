## Stack

- React
- Next.js
  - Full stack React framework
- Node.js
- PostgreSQL
- Prisma
  - ORM for PostgreSQL access
- TypeGraphQL
  - GraphQL framework
- Apollo Server
  - GraphQL server
  - apollo-server-micro
    - Embedded in Next.js server
- React-Query
  - Fetch data from server
- graphql-request
  - GraphQL client
- GraphQL Code Generator
  - Auto-generates code for queries
